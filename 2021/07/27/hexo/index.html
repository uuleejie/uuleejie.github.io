用hexo在github上建立个人blog

用hexo在github上建立个人blog

文章来源: “利用Github搭建自己的个人博客,看完就会的保姆级教程”

作为一名技术爱好者,平时喜欢把自己学习技术的心得或者一些踩坑、易错的过程记录下来，首选的方法是写博客。今天就尝试利用Github来搭建一个自己的个人博客平台。
Github除了可以作为开源代码的交流平台，还提供了搭建博客的功能。目前一些开发者的博客用的就是github搭建的。
本人的个人博客网址：https://uuleejie.github.io

01、环境准备

  • 操作系统: Ubuntu 20.04.2 LTS/64bit/
  • Git程序： git version 2.25.1 (ubuntu自带)
  • node.js: v14.17.3 (ubuntu自带)
  • 安装hexo: npm install hexo -g; 验证安装是否成功：hexo -v
  • 安装hexo依赖：npm install –save hexo-deployer-git

02、 git配置SSH KEY

你应该已经在github上注册了账号。配置SSH key的目的：可以免密的将本地的源码和资源上传到github，无需每次都输入账号和密码。　　　

  1. 先查看本地是否配置好SSH key: $ cd ~/.ssh 如果提示没有这样的文件或目录，表示之前没有配置过SSH KEY。
  2. 配置ssh。$ ssh-keygen -t rsa -C “github账号绑定的邮件地址”，连续回车三次即可。可以看到用户文件夹下生成了一个.ssh文件夹。
  3. 进入到.ssh文件夹，复制id_rsa.pub文件里全部内容。接着打开github主页，点击个人设置->点击左侧的SSH and GPG keys，点击New SSH key，将复制的内容粘贴到key中，title随便起一个就行。点击Add SSH key。
  4. 测试是否成功：ssh -T git@github.com，输入yes，出现提示”You’ve successfully authenticated.but Github not provide shell access. “　就表明SSH key配置成功了。
  5. 配置git账号和密码
    $ git config --global user.name "github用户名"
    $ git config --global user.email "github注册的邮箱"

03、搭建个人博客

1. 先介绍一下hexo的基本命令

$ hexo init
$ npm install


deploy:
type:git
repo:git@github.com:uuleejie/uuleejie.github.io
branch:main


+ 发布到github: $ hexo clean && hexo deploy
+ 用浏览器访问 <https://uuleejie.github.io>。该网址是公网地址，其他人也可以访问。　　　

## 04、编写博客
搭建好博客后，日常需要编写博客并同步到github，可按如下操作。　　　
+ 新建一篇名为：my-first-blog的博客：$ hexo new "my-first-blog",可以看到生成了一个**my-first-blog.md**的文件，通过修改这个md文件去写博客。其中```<!--more-->```的作用是在查看文章的时候，点击more的时候再加载全部的文章内容。修改好后保存。
+ 生成静态网页：$ hexo g
+ 上传到github:$ hexo d
+ 在浏览器中查看　　　

## 05、小结
本文详细介绍了从０到１搭建github博客的过程。希望在今后的学习中不断加深对github pages的认识，用好用精。
